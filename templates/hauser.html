하우저 변환기 사용 방법
🚀 기본 사용 방법
  1. 파일 선택 → 변환할 엑셀 파일 업로드
  2. 변환하기 버튼 클릭
  3. 변환된 파일 다운로드
  4. K~L열 제외하고 하우저 택배의뢰등록양식에 붙여넣기
  5. 하우저 택배일괄등록에서 택배의뢰등록양식 파일 업로드
    등록 실패한 건이 있는지 반드시 확인하고 재등록하세요.
📋 변환 과정
  1. K열의 상품명을 읽어옵니다
  2. 줄바꿈 기준으로 상품명을 분리합니다
  3. 각 상품명에 맞는 하우저 번호를 찾습니다
  4. 결과를 새로운 엑셀 파일로 저장합니다
✅ 변환 조건
  • 모델명: 상품명 괄호 안에 참조 모델명 포함
  • 색상: 상품명에 참조 색상 포함
  • 높이: 상품명 괄호 안에 참조 높이 포함
